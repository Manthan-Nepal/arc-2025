 3. Display Properties

 => Block (take full width,starts on new line)
 => Inline(Only wide as content)

 4. Css positioning

 => relative : Moves element relative to its normal position.

 => absolute : Positioned relative to the nearest ancestor with position: relative;.

 => fixed : Positioned relative to the viewport, stays in place while scrolling.
