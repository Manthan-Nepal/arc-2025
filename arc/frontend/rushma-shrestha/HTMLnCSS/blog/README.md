## Advanced Task: Adding a Dropdown Navbar

### Dropdown Navbar Features
- **Hover activated:** The dropdown menu appears when hovering over the **Categories** menu item.
- **Consistent styling:** Dropdown background matches the header style, ensuring a smooth visual flow.
- **Nested submenu:** The **Sports** category includes a nested submenu with items like Football, Swimming, Basketball, and Volleyball, which appears to the right on hover.
- **User interaction:** Menu items highlight with a light grey background and rounded corners on hover for improved UX.
- **Dropdown positioning:** Positioned just below the header to avoid overlap and maintain clarity.
- **CSS-only implementation:** Dropdown uses pure CSS with the `:hover` pseudo-class and absolute positioning, no JavaScript needed.
