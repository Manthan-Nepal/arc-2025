# HTML Fundamentals – Day 1

Welcome to **Day 1** of my Web Development journey! This folder covers foundational HTML concepts, and two mini-projects built using only HTML5 with proper semantic structure and accessibility in mind.

---

## Objectives

- Set up a modern and efficient development environment.
- Understand and implement semantic HTML structure.
- Create accessible and well-organized static web pages.
- Practice building a portfolio landing page and a blog post layout.

---

## Tools & Setup

To complete the exercises in this folder, you should have the following installed and configured:

- **VS Code** with extensions:
  - Live Server
  - Prettier
  - HTML CSS Support
- **Git** and a **GitHub** account
- Browser **DevTools** for inspecting elements and debugging
- Basic command line usage

---

## Challenges

### Hands-On Exercise

Create a semantic portfolio page using only HTML with no styling. Focus on structure and clean layout using `<section>`, `<article>`, `<header>`, etc.

### Mini-Project

Build a blog post page layout with well-defined main and sidebar sections, using appropriate tags and structure.

### Advanced Challenge

Enhance accessibility using:

- `role` attributes (`navigation`, `main`, `contentinfo`, etc.)
- `aria-label`, `aria-current`, and more
