## What I Did
- Built a blog-like layout using HTML and CSS
- Used `display: flex` and `display: grid` to arrange sections
- Added a responsive navbar and footer
- Included a responsive YouTube iframe for embedding a video
- Used media queries for smaller screens to adjust layout and typography

## Key Concepts Learned

### Flexbox
- Used for navbar (`display: flex; justify-content: space-evenly`)
- Easy to align items in a row and make them responsive

### Grid
- Used for food image layout using `grid-template-columns`
- Helps in arranging items in structured rows and columns

### Positioning
- Used `position: relative/absolute` to place text over the image in the section

### Media Queries
- Adjusted layout for smaller devices (max-width: 600px)
- Resized and repositioned text, images, and footer

## Challenges Faced
- Difficulty in learning Grid
- Aligning section text and images in a responsive way
