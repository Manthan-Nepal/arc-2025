##Summary 
This project is a simple blog-style webpage built using HTML and CSS. It showcases basic layout structures, a responsive navbar, sections for articles, images, videos, and a footer. The site adapts to smaller screens using media queries.

## Features
- Image grid representing food-related blog content
- Embedded YouTube video
- Responsive layout using CSS Grid and Flexbox
- Mobile-friendly design using media queries

## Topics Covered
- HTML5 semantics
- CSS Flexbox & Grid
- CSS positioning (relative, absolute)
- Media queries for responsive design
- Relative units
