# Functions and Modules

## Learning Tasks
1. Define and call functions with various parameter types.
2. Understand scope, default arguments, and return values.
3. Create and import modules and packages.

## Self Learning Tasks
1. Create a utility module with functions for common task operations.
2. Implement functions with positional, keyword, and default argumentss.
3. Use lambda functions and higher-order functions (map, filter, reduce).
4. Explore useful standard library modules like datetime, random, and json.
