# Assignment 2.1 – Interactive Calculator in Python

## Problem Statement
Develop a calculator that:
- Takes **two numbers** as input from the user.
- Allows the user to **choose an operation** to perform (Addition, Subtraction, Multiplication, Division, etc.).
- Performs the selected operation and shows the result.
---

## Features Implemented
- Menu-driven interface
- Basic arithmetic operations and Handles division by zero
- Input validation using `try-except`
- Option to continue or exit
