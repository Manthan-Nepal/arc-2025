# Foundations of Data Science: From Wrangling to Visualization

This repository documents a hands-on journey through the foundational stages of data science, focusing on data wrangling, exploratory data analysis (EDA), and visualization using Python. The content is structured as a series of Jupyter Notebooks, inspired by the [arc-2025](https://github.com/Tech-Manthan-Nepal/arc-2025) curriculum and related resources.

---

## Overview

The primary objective of this codebase is to develop practical skills in:
- **Data Discovery & Cleaning**
- **Data Structuring & Enriching**
- **Exploratory Data Analysis (EDA)**
- **Data Visualization with Matplotlib & Seaborn**

You will learn to reveal the story hidden in data by cleansing, analyzing, and visualizing it before applying any machine learning models.

---

## Topics Covered

1. **Introduction to Data Science & AI**
2. **Data Wrangling**
   - Loading, inspecting, and cleaning data
   - Handling missing values and duplicates
   - Feature engineering (e.g., age groups, BMI categories)
   - Data validation and publishing
3. **Data Visualization Tools**
   - Matplotlib basics: line, scatter, bar, histogram plots
   - Plotting with NumPy arrays and pandas DataFrames
   - Advanced plot customization and saving figures
4. **Exploratory Data Analysis (EDA)**
   - Univariate and bivariate analysis
   - Outlier detection and treatment
   - Correlation analysis and heatmaps
   - Boxplots, scatterplots, and countplots

---

## Learning Outcomes

- **Data Wrangling:** Confidently clean, structure, and enrich real-world datasets using pandas and numpy.
- **Visualization:** Create insightful plots using Matplotlib and Seaborn to communicate data patterns.
- **EDA:** Perform thorough exploratory analysis to understand distributions, relationships, and anomalies in data.
- **Feature Engineering:** Derive new features (e.g., categorical bins) to enhance data understanding.
