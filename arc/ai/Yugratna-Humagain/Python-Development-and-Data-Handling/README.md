# Python Development and Data Handling
## Overview

This repository documents my journey through intermediate Python programming, focusing on object-oriented programming (OOP), scripting, and data processing using NumPy and Pandas. The content is inspired by tutorials and assignments from the [arc-2025](https://github.com/Tech-Manthan-Nepal/arc-2025) repository.

The primary objective was to deepen my understanding of Python's capabilities in data manipulation and analysis, leveraging the power of libraries like NumPy and Pandas.

## Learning Outcomes

### Python Programming

- **Data Structures:** Mastered operations with lists, tuples, sets, and dictionaries.
- **Control Flow:** Implemented complex logic using conditional statements, loops, and comprehensions.
- **Functions:** Utilized various function types, including lambda functions, and understood argument handling with `*args` and `**kwargs`.
- **File Operations:** Performed reading and writing operations on text and JSON files.

### Object-Oriented Programming

- **Classes and Objects:** Designed classes with appropriate attributes and methods.
- **OOP Principles:** Applied encapsulation, inheritance, and polymorphism in projects.
- **Special Methods:** Implemented methods like `__init__`, `__str__`, and utilized static and class methods effectively.

### NumPy

- **Array Creation:** Generated 1D and 2D arrays, exploring their attributes.
- **Indexing & Slicing:** Accessed and modified array elements efficiently.
- **Array Operations:** Performed arithmetic operations, reshaping, concatenation, and splitting of arrays.
- **Broadcasting and Vectorization:** Leveraged these features for efficient numerical computations.

### Pandas

- **Series and DataFrames:** Created and manipulated Series and DataFrames, inspecting data effectively.
- **Data Manipulation:** Handled reading/writing of CSV/JSON files, managed missing data, and performed merging and joining operations.
- **Data Analysis:** Conducted filtering, grouping, aggregation, and computed basic statistics.
