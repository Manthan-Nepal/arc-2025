# Data Wrangling, Visualization & EDA

## Overview

This repository documents my exploration into foundational concepts of Data Science (DS), and practical techniques for handling and analyzing data. The focus was on applying **data wrangling**, **visualization**, and **exploratory data analysis (EDA)** techniques using real-world data such as the **Superstore dataset**.The content is inspired by tutorials and assignments from the [arc-2025](https://github.com/Tech-Manthan-Nepal/arc-2025) repository.

The primary objective was to build a clear understanding of how raw data is cleaned, visualized, and prepared for modeling using Python libraries such as **Pandas** , **Matplotlib** and **Seaborn**.

## Learning Outcomes

### Data Wrangling

- **Handling Categorical Data:** Dropped high-cardinality columns, encoded relevant features.
- **Data Cleaning:** Identified and treated missing values and inconsistencies.
- **Transformation:** Converted raw features into usable formats using encoding methods.

### Visualization with Matplotlib

- **Plot Types:** Created line plots, bar charts, histograms, box plots and scatter plots.
- **Purpose:** Used plots to find trends, detect outliers, and communicate insights visually.

### Exploratory Data Analysis (EDA)

- **Descriptive Statistics:** Used `describe()`, `info()`, and other summary methods.
- **Univariate & Bivariate Plots:** Visualized single-variable and pair-wise relationships.
- **Correlation Analysis:** Explored relationships between features using Pearson correlation.

### Outlier Detection & Treatment

- **Detection Methods:** Applied box plots, z-scores, and IQR method.
- **Treatment:** Removed, capped, transformed, or imputed outlier values as needed.
