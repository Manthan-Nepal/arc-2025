# Superivsed and Unsupervised Learnings on Census Income Data

## Overview

This repository showcases the application of **supervised** and **unsupervised** learning techniques on a real-world dataset to solve a binary classification problem: **predicting whether an individual's income exceeds \$50K/year based on census data**. The project includes **data wrangling**, **exploratory data analysis (EDA)**, **visualization**, and a comparative study of various classification algorithms as well as clustering algorithms.

This work is inspired by foundational topics in Machine Learning, including:
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)
- Decision Trees
- Ensemble Learning methods
- Naive Bayes’ Algorithm
- Principal Component Analysis (PCA)
- Singular Value Decomposition (SVD)
- Clustering & Unsupervised Learning concepts

## Task

> Predict whether an individual's annual income exceeds \$50K/year based on census data.

- **Dataset**: UCI Adult Census Dataset
- **Task**: Binary Classification (`>50K` or `<=50K`)
- **Steps**:
  - Study dataset features and remove irrelevant ones.
  - Perform detailed **EDA** using Seaborn and Matplotlib.
  - Apply **feature engineering** and handle missing data.
  - Implement all relevant **classification algorithms**.
  - Apply **ensemble learning** methods and compare results.
  - Evaluate using **accuracy, precision, recall, F1-score**, and **confusion matrix**.
