# Foundations of Machine Learning

This repository contains comprehensive notes on core mathematical and conceptual foundations of Machine Learning. These topics were covered as part of the ARC-2025 Internship Program.

##  Topics Covered

1. **Introduction to Machine Learning**
   - What is ML?
   - Types of ML: Supervised, Unsupervised, Reinforcement
   - Real-world applications

2. **Linear Algebra for ML**
   - Scalars, Vectors, Matrices
   - Matrix operations: addition, multiplication, transpose
   - Eigenvalues & Eigenvectors

3. **Probability Distribution**
   - PDF, PMF, CDF
   - Normal Distribution: Bell curve, properties, empirical rule
   - Central Limit Theorem

##  Objectives

- Learn foundational mathematical concepts essential for Machine Learning.
- Apply concepts like Linear Algebra and Probability in real-world ML scenarios.
