4. Implement AlexNet from scratch with pytorch and train and evaluate it for Cat Vs. Dog Classification with [this](https://www.kaggle.com/datasets/bhavikjikadara/dog-and-cat-classification-dataset) dataset. Perform relevant image preprocessing with pytorch transforms, create Dataloaders, train the model and generate classification metrics. For this task if you have not GPUs, you can use Kaggle or Google Colab.
5. Perform the same task 4 by adding 2 more CNN layers to AlexNet.
6. Perform the same task 4 by using ResNet as a feature extractor and compare the performance of models from 4, 5, and 6. [Follow learning resource 4]
